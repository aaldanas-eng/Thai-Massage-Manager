# Thai Massage Manager

Sistema de gestión profesional para terapeutas de masaje tailandés.

## Características

### Para Usuarios (Terapeutas)
- Registro con aprobación administrativa
- Gestión de sesiones por spa
- Control de horas trabajadas
- Cálculo automático de impuestos (21%)
- Dashboard con estadísticas
- Filtros por fechas, spa y periodos
- Perfil personalizable

### Para Administradores
- Aprobación de nuevos usuarios
- Gestión de SPAs
- Configuración de precios por hora por usuario
- Visualización de todas las sesiones
- Panel de control administrativo
